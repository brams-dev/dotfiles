# Visual Studio Code Settings
## Instructions

### Windows
```
C:\Users\<USERNAME>\AppData\Roaming\Code\User\settings.json
C:\Users\<USERNAME>\.vscode\extensions
```

## Extensions
* Codesandbox Theme
* Github Markdown Preview
* Markdown Checkboxes
* Markdown Emoji
* Markdown yaml Preamble
* Nomo Dark Icon Theme
